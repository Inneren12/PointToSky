# Icon — adaptive concept

Концепт: указатель в небо + звезда. Минимум деталей, хороший контраст, центр композиции смещён вверх для маски.

Слои:
- `layers/background.svg` — градиент ночного неба
- `layers/foreground.svg` — знак/стрелка и звезда
- `icon-512.svg` — сводный (склеенные слои) для быстрого экспорта

Цвета:
- Night: #0B1E3A → #071224
- Accent: #5DE4FF
- Ink:   #E6F1FF

Экспорт:
```
inkscape -o icon-512.png -w 512 -h 512 icon-512.svg
```
