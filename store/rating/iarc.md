# IARC — подсказки для анкеты (Play Console)

**Тип приложения:** образование/астрономия, справочная утилита.

## Контент
- Насилие: нет
- Сексуальный/эротический контент: нет
- Алкоголь/табак/наркотики: нет
- Оскорбления/брань: нет
- Страх/ужас/шок: нет
- Азартные игры/симуляция: нет
- Пользовательский контент/общение между пользователями: нет
- Обмен локацией с другими пользователями: нет
- Реклама: нет (если добавите позже — отметить)
- Покупки внутри приложения: нет (если появятся — отметить)

**Ожидаемый рейтинг:** Everyone / Для всех (по системам IARC/ESRB/PEGI и др.).
Подробнее о рейтингах и анкете: Play Console Help / IARC.

Перед отправкой перепроверьте соответствие приложения ответам (несоответствия могут привести к снятию с публикации).
